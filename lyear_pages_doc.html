页面说明
当前页面提供了通用的ajax提交方法（处理方法参考DolphinPHP），如果你需要使用，请仔细阅读该说明。

方法采用直接绑定的形式，只需你在需要用到的按钮或者 a 元素上增加类 ajax-post 或者 ajax-get。

ajax-post 提交

1、首先你需要在当前的元素 class 上增加 ajax-post。

2、如果你使用的是 a 标签，则采用 href 指定你要提交到的地址。如果你使用的是按钮，则采用 data-url 指定你要提交到的地址。如果你采用表单，你也可以将地址写在 form 元素的 action 内。

3、还有重要的一点，是你需要指定 ajax 需要处理的表单元素，可以单独是 input textarea select 或者 form，采用 target-form 指定它的 name 值，也可以是唯一的 class 名称。

4、如果你需要在提交前，有弹窗提示，可以在当前元素 class 上增加 confirm，然后 使用 data-tips 指定对对话框的提示语。

需要注意的是，这里做的演示，并没有实际提交到后端，提交后端的代码已经注释，需要您使用时自己打开，这里可以按f12，在console中查看打印输出的信息。
后端返回的数据这里约定格式为：{"code":200,"msg":"处理成功","url":""}

示例一

<a class="btn btn-success m-r-5 ajax-post confirm" href="http://www.lyear.com/doc_test.php" target-form="ids"><i class="mdi mdi-check"></i> 启用</a>

该示例是页面中启用 a 标签的示例，target-form 的值是复选框的 name 名称。

示例二

<form name="testForm" action="http://www.lyear.com/doc_test.php">
  <input type="hidden" name="test" value="2">
  <button class="btn btn-success ajax-post confirm" type="submit" data-tips="测试一下" target-form="testForm-1">示例二</button>
</form>

示例三

<form name="testForm-2"><input type="hidden" name="test" value="2"></form>
<button class="btn btn-success ajax-post confirm" type="submit" data-url="http://www.lyear.com/doc_test.php" target-form="testForm-2">示例三</button>
ajax-get 提交

get 提交，这里不需要指定 target-form，因为要提交的数据内容都在 url 地址里面。

示例一

<a class="btn btn-xs btn-default ajax-get confirm" href="http://www.lyear.com/doc_test.php?id=1" title="" data-toggle="tooltip" data-original-title="删除">
  <i class="mdi mdi-window-close"></i>
</a>

该示例是页面中第一条数据的删除按钮。

示例二

变更数据 
<a class="btn btn-success ajax-get" data-url="http://www.lyear.com/doc_test.php?id=1">变更数据</a>
